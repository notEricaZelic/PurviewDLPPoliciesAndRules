# PurviewDLPPoliciesAndRules

Gets all your enabled Purview DLP Policies and Rules and how they are configured.  Seperates out workloads for easy filtering.  Good for making Power BI Dashboards and drift monitoring.

Credits: Microsoft Docs, M365 Copilot

Connect to Exchange Online Powershell with an appropriate role to run.
